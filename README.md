An automated job alert system built with Python, FastAPI, and Playwright.

## Features
- **Smart Scraping**: Uses Playwright to monitor career pages.
- **Dynamic Alerts**: Users can subscribe to specific departments (R&D, Data, etc.).
- **Persistence**: Powered by SQLite for efficient data management.

## Tech Stack
- Backend: FastAPI
- Scraping: BeautifulSoup & Playwright
- Database: SQLite
- Frontend: Tailwind CSS & Jinja2
